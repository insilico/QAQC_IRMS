# QCIRMS
Quality Control Analysis for Isotope Ratio Mass Spectrometry Data

### To install:

    >library(devtools)
    >install_github("insilico/QCIRMS")  
    >library(QCIRMS)

### Dependencies

```
library(devtools)
install_github('isoverse/isoreader')
```

```
install.packages(c('dplyr', 'pracma'))
```
